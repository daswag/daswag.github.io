> daSWAG release v1.0.0-beta

This is our first beta release for daSWAG v1.

Most important features

  • Create a new daSWAG project and initialize your user management component
  • Add an Api or a Client component to your project.

How to install

This is a beta release, so it is not available on our usual “stable” release channel.

To install daSWAG v1.0.0-beta using NPM:

npm install -g daswag-cli@beta

Help and bugs

If you find any issue with this release, don’t hesitate to:

  • Add a bug on our bug tracker
  • Post a question on Stack Overflow

If the issue you have is an urgent bug or security issue, please:

  • Contact @daswag_project on Twitter
